# Panel_y Parquet データ仕様

---

## 1. フォーマット概要

Panel_y が読み込む標準データ形式は **Apache Parquet** とする。

```
time,   ch_name_1,  ch_name_2,  ...
0.0000, 100.0,      5.0,        ...
0.0001, 101.3,      5.1,        ...
```

---

## 2. カラム仕様

### 2-1. 時間カラム

| 項目 | 仕様 |
|------|------|
| カラム名 | **`time` 固定**（大文字・別名は不可） |
| 単位 | **秒（s）** |
| 型 | float64 |
| 制約 | 等時間間隔であること（下記参照） |
| インデックス | Parquet の row インデックスは使用しない（`index=False` で保存） |

### 2-2. 波形カラム（チャンネル）

| 項目 | 仕様 |
|------|------|
| 判別方法 | `time` 以外の全カラムを波形チャンネルとして自動認識 |
| 型 | **数値型（int / float）のみ**。非数値カラムは読み込み対象外 |
| カラム名 | そのままチャンネル名として UI に表示される |
| 単位 | **表示しない**（カラム名に単位を含める必要はない） |
| チャンネル数 | 1 以上、上限なし（動的に対応） |

---

## 3. サンプリングレート

- `time[1] - time[0]`（先頭2点の差分）からサンプリング周期 Ts を算出する
- サンプリング周波数: `fs = 1 / Ts`
- **等時間間隔を前提とする**。全点で間隔が一定であることを想定

---

## 4. 等時間間隔の前提

- Panel_y は等時間間隔データのみを処理対象とする
- 非等間隔データ（ログ取得のばらつき等）は **前処理ユーティリティで補間・リサンプリングして吸収する**

---

## 5. 非数値データの扱い

- 数値型以外のカラム（文字列、bool 等）は波形チャンネルとして扱わない
- **前処理ユーティリティで除外する**（Panel_y 本体では除外処理を行わない）

---

## 6. 対応する入力形式（前処理変換）

Panel_y 本体は Parquet のみを読み込む。他形式は前処理ユーティリティで変換する。

| 入力形式 | 変換方法 |
|---------|---------|
| **Parquet** | そのまま読み込み（標準形式） |
| **CSV** | `pandas.read_csv()` → 整形 → `to_parquet()` |
| **MATLAB .mat（v7.2以前）** | `scipy.io.loadmat()` → 整形 → `to_parquet()` |
| **MATLAB .mat（v7.3以降）** | `h5py` → 整形 → `to_parquet()` |

---

## 7. 制約まとめ

| 制約 | 内容 |
|------|------|
| 時間カラム名 | `time` 固定 |
| 時間単位 | 秒（s）固定 |
| 等間隔 | 必須（非等間隔は前処理で吸収） |
| チャンネル型 | 数値型のみ（非数値は前処理で除外） |
| 単位表示 | なし |
| メタデータ | Parquet 内には持たない（サンプリングレートは time から算出） |
