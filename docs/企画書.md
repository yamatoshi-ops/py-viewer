## 3. データ仕様

### Panel_y の標準入力形式: Parquet

```
time（秒）, ch_name_1, ch_name_2, ...
0.000,      0.0,       0.0
0.001,      100.0,     5.0
```

- カラム名がそのままチャンネル名として表示される
- メタデータ（単位・サンプリングレート）は別途 JSON で管理（予定）

### 対応入力形式と前処理変換

| 入力形式 | 変換方法 | 備考 |
|---------|---------|------|
| **Parquet** | そのまま読み込み | 標準形式 |
| **CSV** | `pandas.read_csv()` → `to_parquet()` | 変換ユーティリティ内蔵 |
| **MATLAB .mat（v7.2以前）** | `scipy.io.loadmat()` → `to_parquet()` | 変換ユーティリティ内蔵 |
| **MATLAB .mat（v7.3以降）** | `h5py` or `pandas.read_hdf()` → `to_parquet()` | 変換ユーティリティ内蔵 |
