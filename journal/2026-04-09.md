### 作図ファイル形式（.pyc.json）

```json
{
  "version": 1,
  "data_path": "/path/to/measurement.parquet",
  "rows": [
    {"channels": ["ia", "ib"], "ymin": null, "ymax": null, "step_channels": []}
  ],
  "ch_styles": {"ia": {"color": "#1f77b4", "width": 1}},
  "theme": "dark",
  "scroll_zoom": false
}
```

## 判断・設計メモ

- **FFT設定は保存対象外**: 一時的な操作に近く、都度設定で十分と判断
- **ロード時は波形まで一括描画**: `load_plotconfig` が `waveform_row()` を直接呼び出すことで、ロード後に「更新」ボタンを押し直す手間をなくした
- **`allow_duplicate=True` の配置**: `update_waveform_rows` を primary（allow_duplicate なし）、`load_plotconfig` を secondary（allow_duplicate=True）として設計
